# LC Chemistry Notes Website

A multi-page static website for Higher Level Leaving Cert Chemistry.

## Included
- separate pages for the main syllabus topics
- collapsible left navigation
- multiple visual themes
- clear formatting, repaired notation, and helpful support diagrams where needed

## Open locally
Open `index.html` in a browser.

## Theme options
Use the theme control in the top-right corner to switch between:
- Classic
- Editorial
- Study
